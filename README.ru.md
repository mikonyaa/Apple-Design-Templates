# Apple Design Templates

Коллекция практичных SwiftUI-шаблонов для Apple-платформ. Каждый шаблон находится в отдельном репозитории и содержит переиспользуемый Swift Package, запускаемый Xcode demo-проект, тесты, документацию, скриншоты и GIF из реального приложения.

## Доступные шаблоны

### [Liquid Glass Tab Bars](https://github.com/mikonyaa/LiquidGlassTabBars)

Системный, плавающий и morphing tab bar с нативным Liquid Glass на iOS 26 и читаемым fallback для iOS 17–25.

### [Adaptive App Shell](https://github.com/mikonyaa/AdaptiveAppShell)

Единая архитектура навигации: нижние вкладки на iPhone, sidebar и опциональный inspector на iPad.

## Что есть в каждом проекте

- Готовый Swift Package без сторонних runtime-зависимостей
- Demo-проект для Xcode с локальными данными
- Поддержка основных accessibility-настроек
- Тесты и GitHub Actions
- Документация по архитектуре, интеграции и кастомизации
- Учебный prompt для подробного объяснения кода новичку
- Скриншоты, GIF, MIT License и версионированные Releases

## Как устроена коллекция

Этот репозиторий служит каталогом. Код каждого шаблона хранится отдельно, чтобы Issues, Releases, версии и подключение через Swift Package Manager не смешивались. Новые проекты будут добавляться в каталог после полной проверки.

Ошибки следует создавать в репозитории конкретного шаблона. Здесь принимаются только предложения, относящиеся ко всей коллекции.
