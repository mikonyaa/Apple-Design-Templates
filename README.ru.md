# Apple Design Templates

Витрина готовых SwiftUI-шаблонов для Apple-платформ. Код, документация, Issues и Releases каждого проекта находятся в отдельном репозитории, поэтому каталог остаётся компактным и визуальным.

## Шаблоны

| Проект | Для чего подходит | Стабильная версия |
| --- | --- | --- |
| [Liquid Glass Tab Bars](https://github.com/mikonyaa/LiquidGlassTabBars) | Нативный, плавающий и morphing tab bar с Liquid Glass, fallback-поведением и четырьмя темами | `1.0.1` |
| [Adaptive App Shell](https://github.com/mikonyaa/AdaptiveAppShell) | Единая навигация: вкладки на iPhone, sidebar и опциональный inspector на iPad | `1.0.1` |

Каждый шаблон включает переиспользуемый Swift Package, запускаемый Xcode demo-проект, accessibility-поведение, тесты, CI, MIT License и превью из реального приложения.

Подробная архитектура, интеграция, кастомизация и учебный prompt находятся внутри репозитория конкретного шаблона.
